CS-420 Web Server Management

This course provides the student with an introduction to the basics of a Web server management role.
We will cover server installation, configuration, and administration of Web servers.

The student will be exposed to all topics for both Linux and a Microsoft Windows server environment.
They will work with Microsoft Windows 2000 Server and Windows Server 2003,
Internet Information Services (IIS), Apache Web Server, Microsoft SQL Server, MySQL, Microsoft Exchange 2000 Server, sendmail, and more.
